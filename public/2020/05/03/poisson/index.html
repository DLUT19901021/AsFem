Poisson equation

Posted on 2020-05-03 In Examples

A classical Poisson equation can be written out as follow:

∇⋅(σ(ϕ)∇ϕ)=F(ϕ)\nabla\cdot(\sigma(\phi)\nabla\phi)=F(\phi) ∇⋅(σ(ϕ)∇ϕ)=F(ϕ)

with the boundary conditions as follows:

σ(ϕ)∇ϕ⋅n⃗=0on∂ΩN\sigma(\phi)\nabla\phi\cdot\vec{n}=0\qquad\mathrm{on}\quad\partial\Omega_{N} σ(ϕ)∇ϕ⋅n=0on∂ΩN​

ϕ=ϕ0on∂ΩD\phi=\phi_{0}\qquad\mathrm{on}\quad\partial\Omega_{D} ϕ=ϕ0​on∂ΩD​

By choosing a suitable test function δϕ\delta\phiδϕ, we can have the integration of the equation as follow:

∫Ω∇⋅(σ(ϕ)∇ϕ)dV=∫ΩF(ϕ)δϕdV\int_{\Omega}\nabla\cdot(\sigma(\phi)\nabla\phi) dV=\int_{\Omega}F(\phi)\delta\phi dV ∫Ω​∇⋅(σ(ϕ)∇ϕ)dV=∫Ω​F(ϕ)δϕdV

applying the divergence theorem and integrating by parts, one can have the weak form as follow:

∫∂Ωσ(ϕ)∇ϕ⋅n⃗δϕdS−∫Ωσ(ϕ)∇ϕ∇δϕdV−∫ΩF(ϕ)δϕdV=0\int_{\partial\Omega}\sigma(\phi)\nabla\phi\cdot\vec{n}\delta\phi dS -\int_{\Omega}\sigma(\phi)\nabla\phi\nabla\delta\phi dV -\int_{\Omega}F(\phi)\delta\phi dV=0 ∫∂Ω​σ(ϕ)∇ϕ⋅nδϕdS−∫Ω​σ(ϕ)∇ϕ∇δϕdV−∫Ω​F(ϕ)δϕdV=0

Then, by applying the boundary conditions, the residual can be stated as:

RϕI=∫Ωσ(ϕ)∇ϕ∇NIdV+∫ΩF(ϕ)NIdVR_{\phi}^{I}= \int_{\Omega}\sigma(\phi)\nabla\phi\nabla N^{I}dV +\int_{\Omega}F(\phi)N^{I}dV RϕI​=∫Ω​σ(ϕ)∇ϕ∇NIdV+∫Ω​F(ϕ)NIdV

where the superscript III denotes the i-th node of the current element, and where the subscript ϕ\phiϕ represents the related degree of freedom (DoF).

Furthermore, one can have the tangential matrix or the stiffness matrix as follows:

KϕϕIJ=∂RϕI∂ϕJ=∫Ω∂σ(ϕ)∂ϕNJ∇ϕ∇NIdV+∫Ωσ(ϕ)∇NJ∇NIdV+∫Ω∂F(ϕ)∂ϕNJNIdVK_{\phi\phi}^{IJ}=\frac{\partial R^{I}_{\phi}}{\partial\phi^{J}} =\int_{\Omega}\frac{\partial\sigma(\phi)}{\partial\phi}N^{J}\nabla\phi\nabla N^{I}dV\\ \hspace{2px}+\int_{\Omega}\sigma(\phi)\nabla N^{J}\nabla N^{I}dV +\int_{\Omega}\frac{\partial F(\phi)}{\partial\phi}N^{J}N^{I}dV KϕϕIJ​=∂ϕJ∂RϕI​​=∫Ω​∂ϕ∂σ(ϕ)​NJ∇ϕ∇NIdV+∫Ω​σ(ϕ)∇NJ∇NIdV+∫Ω​∂ϕ∂F(ϕ)​NJNIdV

Remember, in order to make the code as simple as possible, the derivative of K is without the “-” sign(according to the mathematical definition of Newton-Raphson, we should obtain the K as: K=−∂R/∂ϕK=-\partial R/\partial\phiK=−∂R/∂ϕ). However, in AsFem, we ignore the “-” sign to make the programming much easier for our users.
